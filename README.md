# Software Testing Course Materials

Course materials repository for the **Software Testing Course**

## 🌐 Live Course

**Access the course here:** https://tanjaq.github.io/Tarkvara-Testimine/

## 📚 Materials

This repository contains:
- `index.html` – Course homepage with lesson overview
- `lesson-01.html` through `lesson-09.html` – Individual lesson presentations
- `lesson-bonus-ci.html` – Bonus lesson on CI/CD pipelines

## 📝 Homework Repositories

Each assignment has its own repository. Fork the relevant repo and open a Pull Request when done.

| Repo | Lessons | Description |
|------|---------|-------------|
| [Bug-Reporting](https://github.com/tanjaq/Bug-Reporting) | 1–3 | Practice writing bug reports using GitHub Issues. Find and report real defects. |
| [Rental-Car](https://github.com/tanjaq/Rental-Car) | 5 | Express.js car rental price calculator. Write unit tests with Jest, refactor code, practice TDD. |
| [User-Register](https://github.com/tanjaq/User-Register) | 6 | Express.js user registration API. Write integration tests with Jest + Supertest, mock external services. |
| [playwright-kriso](https://github.com/tanjaq/playwright-kriso) | 8–9 + Bonus | Playwright E2E tests for Kriso.ee bookshop. Three tasks: flat tests, Page Object Model, CI pipeline. |
